# 💜 ResuMatch AI
### Your Smart Career Partner

## What is this?
An AI powered resume screener that matches 
your resume with any job description and gives
improvement suggestions.

## Features
- Upload PDF resume
- Paste any job description  
- Get match score instantly
- See missing skills
- Get improvement suggestions

## Built With
- Python
- Streamlit
- Scikit-learn
- pdfplumber
